IA utilizada: ChatGPT

Diseña una interfaz web moderna y responsive a partir de los siguientes archivos con las siguientes clases para un "Bloc de Noticias" utilizando un estilo minimalista y profesional.

La página debe incluir:

- Un nabvar estatico y clasico

- Un hero donde tengamos el titulo de nuestra app

- Una cuadrícula de tarjetas de noticias (3 columnas en escritorio, 2 en tablet y 1 en dispositivos móviles). Cada tarjeta debe mostrar un identificador, un título, un resumen de la noticia y un botón "Leer más".

- Utilizar una paleta de colores clara con fondo gris muy suave, tarjetas blancas, color principal lila para botones y elementos destacados, sombras sutiles, bordes redondeados y una tipografía moderna como Inter.

- El diseño debe ser completamente responsive utilizando CSS Grid y Flexbox.

- Agregar efectos hover suaves sobre las tarjetas y botones para mejorar la experiencia del usuario sin exagerar las animaciones.

- Mantener una buena jerarquía visual, espacios amplios entre elementos y un diseño limpio siguiendo buenas prácticas de UX/UI.
